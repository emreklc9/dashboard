# Cordelio Dashboard

Modern, çok temalı ve dil destekli Next.js dashboard uygulaması.

---

## Özellikler

- **3 Renk Teması** — Kırmızı, Yeşil, Mavi (canlı geçiş)
- **Dark / Light Mod** — CSS değişkenleriyle tam tema desteği
- **TR / EN Dil Desteği** — Anlık dil değiştirme
- **Duyarlı Sidebar** — Aktif sayfa vurgulu navigasyon
- **SCSS Modülleri** — Bileşen bazlı stil yönetimi

---
